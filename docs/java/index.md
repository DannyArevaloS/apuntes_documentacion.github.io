# Java

## Introducción a Java

Java es un lenguaje de programación de propósito general, orientado a objetos y diseñado para tener la menor cantidad de dependencias de implementación posible.

## ¿Por qué Java?

Java es uno de los lenguajes de programación más populares del mundo por varias razones:

### Características Principales

- **Portabilidad**: "Write Once, Run Anywhere" (WORA)
- **Orientado a Objetos**: Paradigma de programación basado en objetos
- **Robusto**: Gestión automática de memoria y manejo de excepciones
- **Seguro**: Múltiples niveles de seguridad integrados
- **Multihilo**: Soporte nativo para programación concurrente
- **Alto Rendimiento**: JIT compiler para optimización en tiempo de ejecución

## Ecosistema Java

### Java SE (Standard Edition)
La plataforma base de Java que incluye las APIs fundamentales.

### Java EE (Enterprise Edition)
Extensión de Java SE para aplicaciones empresariales a gran escala.

### Java ME (Micro Edition)
Versión optimizada para dispositivos móviles y embebidos.

## Frameworks Populares

### Spring Framework
El framework más popular para desarrollo de aplicaciones empresariales en Java.

**Características:**
- Inyección de dependencias
- Spring Boot para desarrollo rápido
- Spring MVC para aplicaciones web
- Spring Data para acceso a datos

### Hibernate
Framework de mapeo objeto-relacional (ORM) para Java. [Ver más detalles](hibernate.md)

### Apache Maven
Herramienta de gestión y construcción de proyectos.

### JUnit
Framework para pruebas unitarias en Java.

## Versiones de Java

| Versión | Año | Características Destacadas |
|---------|-----|---------------------------|
| Java 8 | 2014 | Lambdas, Stream API |
| Java 11 | 2018 | LTS, HTTP Client |
| Java 17 | 2021 | LTS, Sealed Classes |
| Java 21 | 2023 | LTS, Virtual Threads |

!!! info "LTS"
    LTS significa "Long Term Support" - versiones con soporte extendido recomendadas para producción.
